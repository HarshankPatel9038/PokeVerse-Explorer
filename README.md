# PokeVerse Explorer

A React + Vite web app for browsing and exploring Pokémon, built with React Router for smooth client-side navigation.

🔗 **Live Demo:** (https://pokeverse-explorer.vercel.app/](https://pokeverse-explorer.vercel.app/)

---

## ✨ Features

- Browse a list of Pokémon with key details (name, type, stats, etc.)
- Client-side routing for individual Pokémon detail pages
- Fast dev/build experience powered by Vite
- Clean component-based React architecture

## 📦 Getting Started

### Prerequisites
- Node.js (v16 or higher recommended)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/HarshankPatel9038/Pokemon.git
cd Pokemon

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
```

The app will be available at `http://localhost:5173` (default Vite port).

### Build for Production

```bash
npm run build
npm run preview
```
